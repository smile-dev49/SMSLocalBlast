SMS LocalBlast — iOS Shortcut Setup

Use the Shortcuts app (iOS 15+) to poll the API and send SMS from your iPhone.

Steps

  1. Open Shortcuts on your iPhone and create a new shortcut.
  2. Add variables: Text actions for YOUR_API_URL (e.g. https://yourserver.com) and YOUR_TOKEN (JWT from login).
  3. Get Contents of URL: POST to [API_URL]/api/messages/claim-next with header Authorization: Bearer [TOKEN].
  4. Parse JSON: Get message from the response.
  5. If message exists: Get id, to_phone, body. Use Send Message to send. Then PATCH /api/messages/[id]/status with {"status":"sent"}.
  6. Add Personal Automation to run on a schedule (e.g. every 2 min).
