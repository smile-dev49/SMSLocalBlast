FAQ

Will carriers ban me?

Smart jitter and rate limiting (~200 msg/hour) help. Stay within your plan's normal usage. Avoid spam content; comply with TCPA/GDPR.

Is this legal?

Yes, when used responsibly. You must have consent to contact recipients. Follow TCPA (US), GDPR (EU), and local regulations.

How do I set it up?

Create a Supabase project, run the SQL scripts, visit /install for the 3-step wizard. Connect your Android device or iOS Shortcut.

Does it support MMS?

Yes. Add an optional Media URL column in Excel. Android supports MMS; iOS uses the Shortcut for SMS.
